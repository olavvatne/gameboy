A gameboy emulator written in JavaScript ES6, and a WIP.

## Getting Started
* Get [Node.js](https://nodejs.org/en/)
* ``` git clone git@github.com:olavvatne/gameboy.git ```
*  ``` npm install ```
*  ``` npm run watch ```
